# md-render

Pure Markdown → HTML rendering crate, extracted from [md-preview](https://github.com/martian17/md-preview). Zero web/async/CRDT dependencies; builds with `--no-default-features` and carries no daemon stack.

Used as a pinned git dependency by `md-preview` and `md-hub`.
